# Hero images

Place your main hero/landing image here.

- **hero.jpg** (or .png) — Used on the homepage hero. Recommended size: 1200×600px or larger, landscape.
- **hero.svg** — Default placeholder included; replace with your photo by adding `hero.jpg` and updating `components/sections/Hero.tsx` to use `HERO_IMAGE = '/images/hero/hero.jpg'`.
